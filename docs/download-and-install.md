# 下载 CSM

---

{: .important }
> CSM 框架需要 LabVIEW 2017 及以上版本。

---

## 方式一：通过 VIPM 安装（推荐）

CSM 通过 VIPM Library 的形式发布，推荐通过 VI Package Manager（VIPM）应用直接搜索并安装。

**安装步骤：**

1. 打开 VI Package Manager（VIPM）
2. 在搜索框中搜索 **CSM** 关键字
3. 找到 `Communicable State Machine` 并点击安装

![VIPM 搜索 CSM](https://nevstop-lab.github.io/CSM-Wiki/assets/img/VIPM%20Search%20CSM.png)

[![VIPM 安装量](https://www.vipm.io/package/nevstop_lib_communicable_state_machine/badge.svg?metric=installs)](https://www.vipm.io/package/nevstop_lib_communicable_state_machine/)

---

## 方式二：通过 GitHub 下载安装包

访问 GitHub Release 页面，下载 `.vip` 安装包后，双击或通过 VIPM 打开安装：

<https://github.com/NEVSTOP-LAB/Communicable-State-Machine/releases>

[![GitHub 发布总量](https://img.shields.io/github/downloads/NEVSTOP-LAB/Communicable-State-Machine/total)](https://github.com/NEVSTOP-LAB/Communicable-State-Machine/releases)

---

## 方式三：通过 Gitee 下载安装包

国内用户也可以通过 Gitee 下载：

<https://gitee.com/NEVSTOP-LAB/Communicable-State-Machine/releases>

---

## 下载 Pre-release 版本

如果想要下载 Pre-release 版本，请访问 GitHub Actions 页面：

[NEVSTOP-LAB/Communicable-State-Machine 的 Action 页面](https://github.com/NEVSTOP-LAB/Communicable-State-Machine/actions/workflows/Build_VIPM_Library.yml?query=branch%3Amain)

**下载步骤：**

1. 点击最新的 Build_VIPM_Library 的 Action
2. 下拉到达 Artifacts 区域，点击 vip 文件下载
3. 解压下载的 zip 文件
4. 双击 vip 文件或通过 VIPM 打开安装
